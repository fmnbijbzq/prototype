学生提交作业流程图

流程说明

本流程图展示了学生完成和提交作业的完整过程，包括查看作业、在线答题、保存草稿和最终提交等关键步骤。

关键功能点：

  • 支持草稿保存功能，防止意外丢失作答内容
  • 截止日期检查，确保按时提交
  • 清晰的提交状态显示
flowchart TD %% 学生完成提交 S1[学生登录系统] --> S2[查看我的作业列表] S2 --> S3[点击某个作业查看详情] S3 --> S4[在线答题页面] S4 --> S5{是否保存草稿?} S5 -- 是 --> S6[保存当前答案为草稿] S6 --> S4 S5 -- 否 --> S7{答题是否完成?} S7 -- 否 --> S4 S7 -- 是 --> S8[点击提交作业] S8 --> S9{是否超过截止日期?} S9 -- 否 --> S10[系统记录提交时间\n更新状态为已提交] S9 -- 是 --> S11[提示已截止，无法提交] S10 --> S12[等待教师批改]
