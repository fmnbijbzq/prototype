lim(x→∞) (1 + 1/x)^x

我知道这个极限的结果应该是e，但是我不明白为什么。希望能得到详细的解答步骤。

王老师 教师

1小时前

李四同学，这是一个经典的极限问题，确实是收敛到自然常数e。

解题思路如下：

1. 首先，我们可以将极限表达式记为y = (1 + 1/x)^x

2. 取对数，ln(y) = x·ln(1 + 1/x)

3. 当x→∞时，1/x→0，我们可以使用泰勒展开：ln(1+t) ≈ t - t²/2 + t³/3 - ... (当t接近0)

4. 代入t=1/x，得到ln(1+1/x) ≈ 1/x - 1/(2x²) + ...

5. 因此，ln(y) ≈ x·(1/x - 1/(2x²) + ...) = 1 - 1/(2x) + ...

6. 当x→∞时，ln(y)→1，所以y→e¹=e

我们也可以用换元法，令n=x，则原式变为：lim(n→∞) (1 + 1/n)^n，这是e的定义式。
